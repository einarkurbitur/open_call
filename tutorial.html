A small tutorial on Photoshop actions

Open your Photoshop, create a new file and go to Window->Actions
Warning: if you start with an image instead of a blank canvas the action might not work properly on other computers, unless you attach the image with it. Click on the new folder button and give your folder a name. Click on the new action button and give your action a name. To enable tool recording (recording brushes etc.), press the button in the top right corner of the action window and select "Allow Tool Recording"
 To record your action press the circle button at the bottom of the action window.
Warning: the undo/redo functioins are not recorded so if you need to undo something, stop the recording and delete your last entry from the action list. To stop recording press the square button at the bottom of the action window. To play the action press the play button at the bottom of the action window.
Important: Open a new blank file and test out your action to make sure it works. To change the playback speed, press the button in the top right corner of the action window and select "Playback Options...". From there you can choose to either play it step by step or instantaneous.
Important: This is not saved in the action so if you have a specific method you want your action to be played, please attach a .txt file with specifications. When your action(s) is ready and you have tested it, press the folder with the action(s) in it. Then press the button in the top right corner of the action window and select "Save Actions..."
Beware you can only export sets, not independent actions. Name your action "title_of_your_work_your_name.atn" and upload it on the main page. If you want your action to be played/used in a specific manner, please attach a .txt file with the action along with all extra files. It is preffered that you combine all your files in one .zip file project by: Einar Andersen and Florian van Zandwijk
